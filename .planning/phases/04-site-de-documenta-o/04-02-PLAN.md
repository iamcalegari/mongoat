    - Registrar um pre-hook via `Model.pre(METHODS.INSERT, (ctx) => { ctx.document.field = ... })` — enfatizar a assinatura `ctx` (NÃO `this`-bound) e que `ctx` expõe `document`/`options`/`filter`/`update`/`result`/`model`/`method` conforme o método.

    - Deixar claro que a validação é server-side (MongoDB-native), não client-side.
